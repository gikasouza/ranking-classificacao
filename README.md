# Ranking Classificação

Projeto em Python que gera uma lista de classificação.

## 📌 Funcionalidade
O programa permite criar e exibir um ranking de nomes ou pontuações.

## ▶️ Como executar

1. Baixe o projeto ou clone o repositório
2. Abra o terminal na pasta do projeto
3. Execute:

```bash
python ranking.py
```

## 💻 Tecnologias
- Python
