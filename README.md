# 🏆 Ranking Classificação

Projeto simples em Python para criação e exibição de um ranking de nomes ou pontuações.

---

##  Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar lógica de programação em Python, permitindo organizar e exibir uma lista de classificação de forma simples.

---

##  Funcionalidades
✔ Adicionar participantes  
✔ Organizar ranking  
✔ Exibir classificação  

---

##  Como executar

1. Clone o repositório:
```bash
git clone https://github.com/gikasouza/ranking-classificacao.git
```

2. Acesse a pasta:
```bash
cd ranking-classificacao
```

3. Execute o programa:
```bash
python ranking.py
```
